Control de temperatura

Temperatura confort:

La temperatura de confort es la temperatura objetivo cuando se selecciona el modo "Confort" (símbolo del sol). Este modo mantiene caliente la casa, pero consume mas energía. Se recomienda el usos durante el día.


Temperatura ahorro :

La temperatura de ahorro es la temperatura objetivo cuando se selecciona el modo "Ahorro" (símbolo de la luna). Este modo no mantiene tan caliente la casa, pero no consume tanta energía. Se recomienda el uso durante la noche o cuando durante el día no estamos en la casa.


Temperatura viaje :

La temperatura de viaje es la temperatura objetivo cuando se selecciona el modo "Viaje" (símbolo de la maleta). Este modo mantiene la temperatura al minimo necesario para que no estropee las cañerias o ningun equipo de la casa. Se recomienda el uso cuando abandonamos la casa por periodos de tiempo largos (vacaciones, ...).


Limitador temperatura

Para evitar cindesación de la humedad los dias muy frios, se recomienda mantener activado el limitador de calefacción para que se ajuste automáticamente de acuerdo a la temperatura exterior. Cuando se usa el limitador de temperatura, la temperatura se limita de la suigiente forma:

  • 20ºC cuando la temperatura exterior esta entre 5-10ºC
  • 19ºC cuando la temperatura exterior esta entre 0-5ºC
  • 18ºC cuando la temperatura exterior es menor de 0ºC
